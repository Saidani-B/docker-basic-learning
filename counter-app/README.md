# counter-app
Simple flask app that counts web site visits and stored in a default Redis backend.
# How it works:
1-To get started first install Docker by following the link to the official download page "https://docs.docker.com/get-started/get-docker/"
2-Once you have Docker running, build and run your application using: "docker comse up -d".
3-Finnaly to make sure it-s working just open a web browser and go to localhost on port 8888 and you should see the hit counter increase every time you visit
